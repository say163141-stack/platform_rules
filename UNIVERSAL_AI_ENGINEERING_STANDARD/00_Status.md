Purpose
-------

This repository defines the engineering standard governing the design,
implementation, validation, operation and continuous evolution of AI-driven
software systems.

The standard is implementation-independent.

It is intended to be applicable to systems built with:

- n8n
- LangGraph
- CrewAI
- Temporal
- AutoGen
- OpenAI Agents SDK
- custom orchestration frameworks

This document is the normative source of truth.

All project-specific documentation derives from this standard.

3. Approval
4. Version Increment
5. Changelog Entry
